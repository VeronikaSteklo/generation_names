# Long Short-Term Memory (LSTM)

Модуль реализует модель на основе рекуррентной нейронной сети с долгой краткосрочной памятью (LSTM) для генерации
заголовков по входному тексту.

Архитектура LSTM решает проблему затухающего градиента, характерную для классических RNN, благодаря механизму ворот (
input, forget, output gates) и ячейке памяти. Это позволяет модели захватывать долгосрочные зависимости в тексте.

Модель последовательно предсказывает слова названия на основе:

1. Контекста из исходного текста, закодированного через embeddings;
2. Скрытого состояния LSTM, аккумулирующего информацию о предыдущих токенах;
3. Уже сгенерированных слов на каждом шаге декодирования.

Пример генерации:

**Входной текст:**
> *«Каждый из нас понимает очевидную вещь: консультация с широким активом предопределяет высокую востребованность
экспериментов, поражающих по своей масштабности и грандиозности. Есть над чем задуматься: некоторые особенности
внутренней политики ограничены исключительно образом мышления. Таким образом, убеждённость некоторых оппонентов создаёт
необходимость включения в производственный план целого ряда внеочередных мероприятий с учётом комплекса экономической
целесообразности принимаемых решений.»*

**Сгенерированный заголовок:**
> онегин.. будды шакьямуни шакьямуни того, московский московский пациент пациент озолина озолина аввакума, хочется
> хочется хочется хочется хочется хочется хочется хочется

**Входной текст:**
> *«абоба»*

**Сгенерированный заголовок:**
> 06

## Особенности архитектуры

- **Encoder-Decoder структура**: входной текст кодируется через embedding-слой, после чего LSTM генерирует заголовок
  токен за токеном;
- **Masked Loss**: при обучении игнорируются паддинг-токены, что позволяет работать с последовательностями разной длины;
- **Early Stopping**: обучение останавливается автоматически, если loss на валидации не улучшается в течение заданного
  числа эпох;
- **Teacher Forcing**: на этапе обучения модель получает корректные предыдущие токены, что ускоряет сходимость.
