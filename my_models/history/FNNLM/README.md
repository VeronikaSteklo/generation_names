# Feedforward Neural Network Language Model ( FFNNLM )

Модуль реализует Feed-Forward Neural Network Language Model (FNNLM) для генерации заголовков по входному тексту.
Модель последовательно предсказывает слова названия на основе:

1. Контекста из исходного текста;
2. Уже сгенерированных слов.


## Результаты

| Метрика        | Значение |
|:---------------|:---------|
| **Train Loss** | 4.891    |
| **Val Loss**   | 5.204    |

Пример генерации:

**Входной текст:**
> *«Каждый из нас понимает очевидную вещь: консультация с широким активом предопределяет высокую востребованность  экспериментов, поражающих по своей масштабности и грандиозности. Есть над чем задуматься: некоторые особенности  внутренней политики ограничены исключительно образом мышления. Таким образом, убеждённость некоторых оппонентов создаёт  необходимость включения в производственный план целого ряда внеочередных мероприятий с учётом комплекса экономической целесообразности принимаемых решений.»*

**Сгенерированный заголовок:**
> Модель ничего не сгенерировала

**Входной текст:**
> *«абоба»*

**Сгенерированный заголовок:**
> 06 гунин

В результате модель на больших и сложных текстах выдает пустой результат. Это происходит из-за фиксированного окна в 8 слов и использования Mean Pooling, специфические токены сложного текста превращаются в усредненный «шум». В результате вероятность стоп-токена `<END>` становится максимальной, и модель завершает генерацию, не успев её начать.
