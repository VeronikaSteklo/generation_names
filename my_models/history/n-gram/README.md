# N-gram model

Модуль реализует модель основанную на n-граммах для генерации заголовков по входному тексту.
Модель последовательно предсказывает слова названия на основе:

1. Контекста из исходного текста;
2. Уже сгенерированных слов.

## Результаты

Средняя `METEOR`-оценка на валидационном наборе: 0.0000

Пример генерации:

**Входной текст:**
> *«Каждый из нас понимает очевидную вещь: консультация с широким активом предопределяет высокую востребованность
экспериментов, поражающих по своей масштабности и грандиозности. Есть над чем задуматься: некоторые особенности
внутренней политики ограничены исключительно образом мышления. Таким образом, убеждённость некоторых оппонентов создаёт
необходимость включения в производственный план целого ряда внеочередных мероприятий с учётом комплекса экономической
целесообразности принимаемых решений.»*

**Сгенерированный заголовок:**
> окрыленный странноно фельдъегерем жыты новость порами уссурийских

В результате модель формирует последовательности слов, которые не связаны по смыслу с содержанием текста. Это
подтверждает ограниченность n-граммного подхода в задачах семантической генерации.
