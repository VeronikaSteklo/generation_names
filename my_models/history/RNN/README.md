# Recurrent Neural Network (RNN)

Модуль реализует классическую рекуррентную нейронную сеть (Vanilla RNN) для генерации заголовков по входному тексту.

В отличие от LSTM, классическая RNN использует простое рекуррентное соединение: скрытое состояние на каждом шаге
вычисляется как нелинейная функция от текущего входа и предыдущего скрытого состояния. Это базовая архитектура, которая
позволяет оценить, насколько достаточна простая рекуррентность для задачи условной генерации названий.

Модель работает по следующему принципу:

1. Входной текст пропускается через embedding-слой и кодируется с помощью RNN в финальное скрытое состояние;
2. Это скрытое состояние используется как начальный контекст для генерации заголовка;
3. На каждом шаге декодирования RNN предсказывает следующее слово названия, опираясь на уже сгенерированные токены и
   скрытое состояние от текста.

## Особенности архитектуры

- **Encoder-only передача контекста**: финальное скрытое состояние от текста передаётся в RNN-декодер как начальное
  состояние (`h_0`);
- **Simple RNN cell**: используется базовый рекуррентный слой (`nn.RNN`) без механизмов ворот, что делает модель
  уязвимой к проблеме затухающего градиента;
- **Masked Loss**: при обучении игнорируются паддинг-токены, что позволяет работать с последовательностями разной длины;
- **Early Stopping**: обучение останавливается автоматически, если loss на валидации не улучшается в течение заданного
  числа эпох.
